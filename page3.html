LookOut 

Emprendimiento social, el cual entrega un servicio a los colegios para ayuda a los profesores, mediante inteligencia artificial, a detectar a tiempo las distintas situaciones que puedan estar afectando a un escolar durante su desarrollo. Ya sea por bullying, aislamiento, depresión, ansiedad u otros.

LIVE DEMO

Recolección de imágenes

Las imágenes son recolectadas mediante una cámara en la sala de clases, encriptadas y enviadas al servidor local.

Análisis de datos

Análisis de imágenes, mediante reconocimiento facial y machine learning.

Visualización de information

Entrega de información importante mediante una plataforma web interactiva. 

¿Cómo funciona?

  • Indicador interpersonal
  • Indicador intrapersonal

El sistema puede medir cuando una persona está interactuando o está sola. El color rojo significa soledad, por el contario, el azul indica interacción con otra persona.

Mediante reconocimiento facial, es posible detectar las emociones de cada persona. Se puede identificar la emoción en la parte superior del recuadro facial.

Beneficios

Monitoreo constante

El sistema es capaz de analizar una imagen cada 5 segundos, durante todo el periodo escolar. 

Información precisa

Se muestra la información necesaria de cada alumno, con foco sólo en los casos críticos dentro del aula.

Integración de datos

Abre la posibilidad para integrar datos escolares relevantes como antecedentes académico, familiares o psicológicos.
